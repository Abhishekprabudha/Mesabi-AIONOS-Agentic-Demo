# Mesabi AIonOS Demo Script

## Opening
"Mesabi is at the perfect moment to become AI-native from day one. Historical operations data is still limited, so the first wave should not depend on mature predictive history. It should use schedules, SOPs, commissioning packs, OEM manuals, engineering constraints, synthetic scenarios and human approvals to create immediate value."

## Flow

1. Mission Control
   - Show the operating loop: Detect → Diagnose → Simulate → Recommend → Approve → Learn.
   - Press **Start Mesabi scenario**.
   - Explain that every agent produces either a decision, a deliverable or a measurable KPI improvement.

2. Pit-to-Pellet Chain
   - Explain the agent layer across construction readiness, geology, mine/fleet, beneficiation, pellet plant, ESG and shipping.
   - Highlight that this complements SAP/FMS/SCADA/LIMS/CMMS/GIS instead of replacing them.

3. Video Twin
   - Play the uploaded mining video.
   - Point out overlays for haul lane, stockpile boundary, haul unit, loader and dust plume proxy.
   - Explain that video is a signal source, not the full system.

4. Readiness Tower
   - Click **Generate readiness pack**.
   - Explain the readiness pack: A/B/C punch list, RACI, schedule delta, permit matrix, spares exceptions and approval route.

5. Pellet Quality
   - Show how the processing agent explains off-spec risk using P80, silica and moisture signals.
   - Emphasize simulation-first logic before live operational history matures.

6. Geology AI
   - Show QAQC exception register, model confidence heat map, resource scenario book and drill program optimizer.
   - Explain that Mesabi gets orebody confidence before steady-state production data exists.

7. Reliability + ESG
   - Show ranked actions across conveyor reliability, fleet queue, water, dust and carbon evidence.
   - Emphasize owners, due dates, priority and value.

8. 90-Day PoV
   - Explain the six phases from value workshop to AI war-room operating model.
   - Close with approval request: three low-data immediate agents plus two ramp-up pilots.

## Closing line
"AIonOS can help Mesabi start faster, ramp safer and run every ton with intelligence by building the operating brain before operational habits become fixed."
